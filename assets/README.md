# Dori 朵荔网站素材说明

把素材放在本文件夹：`assets/`。

## 核心素材

- `dori-hero.png`：首页右侧 Dori 主视觉。
- `dori-character.png`：角色设定板块的 Dori 角色视觉。
- `overview-visual.png`：项目概览板块视觉。
- `little-fox.png`：剧情入口与视觉体系中的小狐狸情绪残响。
- `paper-cup-phone.png`：剧情入口中的纸杯传声筒。
- `omera-world.png`：Omera 世界观视觉。
- `echora-pendant.png`：回响坠设计图，也用于项目亮点、剧情入口与视觉体系。
- `mood-dori.png`：视觉体系中的角色视觉 moodboard。
- `mood-dreamcore.png`：视觉体系中的真实梦核 / 青春 MV 截图感。
- `mood-omera.png`：视觉体系中的 Omera 空间视觉。
- `mood-color.png`：视觉体系中的材质与色彩。
- `dori-demo.mp3`：音乐方向板块的 Dori 音乐 Demo。
- `dori-trailer.mp4`：概念预告 / AI MV 区域的视频。
- `trailer-cover.png`：概念预告 / AI MV 区域的视频封面。

素材不存在时，网页会自动显示渐变占位图或占位框。

## 本地打开

直接双击项目根目录里的 `index.html`，或在浏览器打开：

```text
file:///Users/lucylu/Documents/condex/index.html
```

## 部署到 GitHub Pages

1. 新建一个 GitHub 仓库。
2. 上传 `index.html`、`style.css`、`script.js` 和整个 `assets/` 文件夹。
3. 进入仓库的 Settings。
4. 打开 Pages。
5. Source 选择 Deploy from a branch。
6. Branch 选择 `main` 和 `/root`。
7. 保存后等待 GitHub 生成访问链接。
